# CS4361 – 3D Planetary System Simulation

A real-time interactive 3D simulation of a planetary system, built for CS4361 (Summer 2026), Team 2.

## Team
Rumaisa Asad · Carlos Antonio Esquivel · Yousef Ahmed · Prethel Alam

## Requirements
- Unity 6000.3.19f1 (LTS) — install via Unity Hub; all team members must use this exact version
- Visual Studio Code with the C# extension
- Git and Git LFS

## Setup
1. Install the tools listed above.
2. Run `git lfs install` once on your machine (first time only).
3. Clone the repo:
   git clone https://github.com/Carlos-A-Esquivel/CS4361-Planetary-System.git
4. Open Unity Hub → Add → select the cloned folder.
5. Open the project (Unity may take a few minutes to import the first time).

## Workflow
- Work on feature branches, not directly on main.
- Merge via pull requests.
- Avoid two people editing the same scene file at once — Unity scenes are hard to merge.
- Pull the latest main before starting new work.
